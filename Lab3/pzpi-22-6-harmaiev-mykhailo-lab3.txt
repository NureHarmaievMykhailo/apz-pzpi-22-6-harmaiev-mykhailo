3 ОПИС ВИКОНАНОЇ РОБОТИ
Веб-клієнт системи SmartRoad було розроблено на основі ASP.NET Core Razor Pages. Для інтерактивного функціоналу застосовувався JavaScript та вбудовані можливості Razor, що забезпечують динамічне оновлення вмісту сторінок.
Фронтенд забезпечує такі основні можливості:
•	авторизацію та автентифікацію користувачів із різними ролями (Admin, Worker, User);
•	перегляд стану дорожніх ділянок у вигляді списків та карт;
•	отримання актуальних сповіщень про критичні ситуації;
•	інтеграцію з REST-API серверної частини для відображення сенсорних даних у реальному часі;
•	панель адміністратора для керування користувачами, сенсорами та дорожніми ділянками.
Для клієнтської частини системи було побудовано UML-діаграму прецедентів, яка демонструє основні сценарії взаємодії користувачів із системою. Також було створено UML-діаграму компонентів, що показує взаємозв’язки між інтерфейсними модулями веб-клієнта та серверною частиною, та UML-діаграму пакетів, яка описує організацію коду клієнта (див. Додаток Б).

4 ВИСНОВОК
У ході виконання лабораторної роботи було реалізовано веб-клієнт системи SmartRoad, який забезпечує зручний інтерфейс взаємодії користувачів із серверною частиною. Було використано технології ASP.NET Core Razor Pages, CSHTML, CSS та JavaScript для побудови структури, дизайну й інтерактивного функціоналу.
Система підтримує авторизацію та автентифікацію користувачів за допомогою JWT, надає можливість відображення стану дорожніх ділянок, перегляду сповіщень та адміністрування даних. Веб-клієнт інтегрується з REST API серверної частини, що дозволяє працювати з даними сенсорів у режимі реального часу.
Також було побудовано UML-діаграми (прецедентів, компонентів, пакетів), які відображають основні сценарії роботи користувачів, структуру клієнтської частини та організацію коду. Це забезпечує наочність архітектури та спрощує подальший розвиток системи.
У результаті виконаної роботи створено працездатний прототип веб-клієнта, який буде розширено додатковими модулями (мобільною адаптацією) і стане важливою складовою комплексної системи моніторингу стану доріг.
